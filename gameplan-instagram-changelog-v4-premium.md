# GamePlan — Instagram Changelog v4 Premium
**Editorial carousel. Nothing-style restraint. Small-phone legibility first.**

---

## CREATIVE DIRECTION

This version should feel:
- quieter
- sharper
- more premium
- less promotional
- more like a product editorial than a school poster

Reference mood:
- Nothing-style composition
- black / deep navy field
- off-white type
- burnished gold used sparingly
- mono labels
- oversized headlines
- strict spacing
- almost no decorative clutter

Do not make it loud.
Do not use gradients unless extremely subtle.
Do not stack too much copy on any slide.

---

## FORMAT RULES

- Size: `1080 x 1350`
- Safe text width: `820 px`
- Safe top/bottom margin: `120 px`
- Headline max: `3 lines`
- Body max: `5 short lines`
- Use one UI crop only when needed
- Every slide must still read clearly at arm’s length on a phone

---

## TYPE SYSTEM

- Eyebrow / labels: small mono, uppercase, tracked out
- Main headline: very large display, 1 to 3 lines max
- Body: short, clean, high contrast
- CTA: small mono or restrained sans

Recommended proportions:
- Eyebrow: `26–32 px`
- Headline: `110–150 px`
- Body: `38–46 px`
- Fine note: `24–28 px`

Use less text than you think.

## DESIGN NOTES FOR THE CREATIVE

### 1. Use emptiness
The premium feel should come from restraint, not decoration.

### 2. Use fewer screenshots
If the UI is too small, do not use it.
Typography beats unreadable UI every time.

### 3. Keep the palette disciplined
- background: near-black or deep navy
- text: warm white
- accent: muted gold only

### 4. Avoid common carousel mistakes
- no dense paragraphs
- no tiny labels
- no fake glass
- no bright gradients
- no over-framing everything in cards

### 5. Best slide treatment
- slide 1, 6, and 8 should be mostly type
- slide 3 and 4 can carry the strongest UI crops
- slide 7 should feel systemic, not technical
